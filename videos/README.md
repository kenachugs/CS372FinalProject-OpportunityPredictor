# videos/

Place your two project videos in this folder after recording:

- `demo.mp4` — 3–5 minute demo video for a non-specialist audience (see `docs/demo_video_script.md` for the script)
- `technical_walkthrough.mp4` — 5–10 minute technical walkthrough (see `docs/technical_walkthrough_script.md`)

The `README.md` at the project root includes a **Video Links** section; after you upload the videos to YouTube / Drive / Panopto, paste those links into that section so graders can reach them.
